- `docker run -it hello-node-world bash`
docker run → φτιάξε νέο container από image και τρέξ’ το
-it → (keyboard) και terminal (shell)
hello-node-world → το image που έφτιαξες
bash → η εντολή που θα τρέξει μέσα στο container
Τι γίνεται πρακτικά:
Δημιουργείται καινούργιο container. Από το image hello-node-world και ανοίγει bash shell
